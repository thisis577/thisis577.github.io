產品的五個狀態

理想狀態（Ideal State）

空白狀態（Empty State）

  • User 清除導致全空
  • 找不到資料

錯誤狀態（Error State）


不要嚇到 User

局部資料狀態（Partial State）

載入中狀態（Loading State）
