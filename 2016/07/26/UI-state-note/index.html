Jul 26, 2016# UI# reading

產品的五個狀態

理想狀態（Ideal State）

空白狀態（Empty State）

  • 首次使用
  • User 清除導致全空
  • 找不到資料

錯誤狀態（Error State）


不要嚇到 User

局部資料狀態（Partial State）

  • 讓 User 知道離完美狀態有多遠
  • 如「個人檔案完成度」

載入中狀態（Loading State）

  • 可先載入框架，讓 User 知道這一頁會長什麼樣子

不要讓 User 撞牆，要他有用的建議或不一樣的途徑

Ref: 如何修正壞UI
