很難懂的 Jade

Hexo-theme-apollo 模版是以 Jade 寫的，與官方使用的 ejs 不同
但只需要另外安裝 Jade 外掛 hexo-render-jade

其實只需要一行指令

如此一來只要下 hexo server 就會自動幫你 compile
也可以另外插入 css 去修改了


凡是有修改 *.jade 建議關掉重開 hexo 比較好，以確保有吃到變更。
