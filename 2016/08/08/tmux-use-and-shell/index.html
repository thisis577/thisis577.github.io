tmux 常用指令 + 自己寫 shell

Aug 8, 2016# shell# tmux

常用指令

tmux ls 列出目前所有 session
tmux a -t <session-name> 跳至該 session
tmux a -dt <session-name> 將其他 client 踢掉

tmux 的操作都是 ^B 開頭：

  • ^B+<number> 切換 tmux window (現在所在的窗會以*標記
  • ^B+C 建新的窗(window)
  • ^B+D 維持 tmux 背景執行並離開
  • ^B+, 重新命名視窗
  • ^B+$ 命名目前 session
